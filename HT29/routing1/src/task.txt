1. Создайте приложение React, которое имеет две страницы: "Главная" и "О нас". Используйте маршрутизацию (routing) с помощью React Router для переключения между страницами.
2. На странице "Главная" должно быть простое приветственное сообщение и ссылка на страницу "О нас". На странице "О нас" должна быть краткая информация о вашей команде/компании и ссылка на страницу "Главная".
3. Когда пользователь переходит между страницами, должен изменяться URL и отображаться соответствующий компонент.
4. *Если пользователь вводит несуществующий URL, должна отображаться страница "404".
