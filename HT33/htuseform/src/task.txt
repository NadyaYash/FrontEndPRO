Предположим, что у вас есть веб-приложение, которое позволяет пользователям создавать свои профили. Вы хотите использовать библиотеку react-hook-form для управления формой создания профиля.
Задача: Создайте форму для создания профиля пользователя, используя библиотеку react-hook-form. Форма должна содержать поля "имя", "фамилия", "адрес", "электронная почта" и "номер телефона". Все поля должны быть обязательными. Для поля электронной почты настройте валидацию - введенное значение должно соответствовать формату электронной почты. При сабмите формы формируется объект с данными и выводится в консоль.
Стилизуйте форму по своему желанию.
